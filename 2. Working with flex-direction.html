Flexbox: Working with flex-direction

flex-direction is important because it sets the axis when we're using flexbox

In Flexbox, we have both the main axis and the cross axis

When we run into things like justify content and align items, which entirely depend on flex-direction

flex-direction: row;

Main Axis: left ========> right

This is the default flex-direction of any flex container- flex elements will stack right beside each other and will span vertically to hit the height of whatever the container is set to
